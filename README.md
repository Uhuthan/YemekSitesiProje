# 🍔 Yemek Sipariş Sitesi Projesi

## 📝 Açıklama
Bu proje; HTML5, CSS3 ve JavaScript kullanılarak geliştirilmiş, kullanıcıların menüdeki yemekleri (çorba, pide, lahmacun vb.) inceleyebildiği ve seçtikleri ürünleri sepetlerine ekleyebildiği dinamik bir web arayüzü çalışmasıdır.

## 🚀 Özellikler
* **Kategorize Menü:** Ürünlerin çorba, pide, lahmacun gibi türlere göre listelenmesi.
* **Dinamik Sepet Sistemi:** JavaScript kullanılarak ürünlerin sepete eklenmesi ve sepet içeriğinin anlık yönetimi.
* **Responsive Tasarım:** CSS ile farklı ekran boyutlarına uyumlu arayüz.

## 🛠 Kullanılan Teknolojiler
* **Yapı:** HTML5
* **Tasarım:** CSS3
* **Mantık/Etkileşim:** JavaScript (Vanilla JS)
